# VLAN Management

VLAN-related data may surface indirectly through synchronized interface and network information, but Proxbox is not currently a dedicated VLAN management plugin.

## Current Position

- Proxbox focuses on discovery and synchronization from Proxmox into NetBox.
- VLAN and interface data are represented through the normal NetBox models once synchronized.
- Direct VLAN lifecycle management from the Proxbox UI is not a primary feature today.
